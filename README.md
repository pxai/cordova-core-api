# cordova-core-api
Samples using cordova core api
# Platforms
# Build and run cycle
Add to platform

Build the app

```
cordova build browser
```

# Run the app
```
cordova run browser
```
# Add plugins
```
cordova plugin add https://github.com/apache/cordova-plugin-contacts.git
```
